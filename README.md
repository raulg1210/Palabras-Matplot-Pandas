# Análisis de Frecuencia de Palabras

Este es un proyecto simple que cuenta la frecuencia de palabras en un archivo de texto y visualiza las palabras más frecuentes en un gráfico de barras.

## Requisitos

- Python 3
- Pandas
- Matplotlib
